# Contributing to HILS
Most of the details are in `README.md`.

## Getting started:
1. Clone the repo.
2. `cd` into the repo.
3. `cd` into `html` and create a `config.php`. Best way is to `cp config.example.php config.php`.
4. Edit `config.php`. Details are in there.
5. Launch a dev server. I use `php -S 127.0.0.1:8000` in the `/html/` directory.
6. Go to `localhost:8000`. It should send you over to `/setup`. If not go to `localhost:8000/setup`.
7. Follow the wizard.
8. You should now be fully set up.

## Committing changes.
Fork the repo, make a new branch in your fork and do some code. Then submit a pull request with the details of your changes and I'll take a look.

Thanks in advance! Feel free to reach out to me on my Mastodon (see `README.md`).
